# 기여 가이드
이 레포에 기여하려면 아래 절차를 따른다.

## 이슈
버그 신고나 기능 제안은 이슈로 먼저 등록한다.

<br>

## PR
1. 레포를 fork한다.
2. 작업 브랜치를 생성한다.
3. 변경 사항을 커밋한다.
4. PR을 보낸다.

<br>

## 커밋 메시지
`[prefix] 내용` 형식으로 작성한다.
- `feat`: 새 기능
- `fix`: 버그 수정
- `docs`: 문서
- `refactor`: 리팩토링
- `test`: 테스트
- `chore`: 기타
